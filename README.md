# Вариант 16
1) Определить структурный тип, описывающий расписание маршрутов электропоездов (название маршрута, пункт отправления, пункт прибытия, время отправления, время прибытия, количество вагонов, тип электропоезда: товарный, пассажирский, ремонтный и др.). Заполнить структурный массив 20-ю записями. Переписать из исходного массива в другой массив, информацию только о тех пассажирских маршрутах, что идут в туристические города. Затем новый массив отсортировать по пункту отправления вагонов. Вывести все данные по конкретному электропоезду. Вывести 3 самых коротких пассажирских поезда. Реализовать функцию изменения данных о маршруте. Реализовать вывод отфильтрованных данных в виде оберточной функции.
